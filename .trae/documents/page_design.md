# Page Design - Delicia Kebab 2

## 1. Visual Style
- **Theme:** Dark Mode.
- **Background:** `#1a1a1a` (Charcoal) or `#000000` (Black).
- **Primary Accent:** `#FFC107` (Amber/Yellow) - Used for buttons, highlights, and prices.
- **Secondary Accent:** `#EF4444` (Red) - Used for urgency, sales, or secondary buttons.
- **Text:** White or light gray (`#f3f4f6`) for readability against dark backgrounds.

## 2. Layout Breakdown

### Header (Navbar)
- **Left:** "Delicia Kebab 2" Logo/Text.
- **Right:** Navigation links (Home, Offers, Menu, Contact) + Halal Badge (Gold/Green styled).
- **Mobile:** Hamburger menu.

### Hero Section
- **Background:** High-quality dark image of a pizza or kebab.
- **Content:** Centered text.
    - H1: "Delicia Kebab 2"
    - Subtitle: "Sabor Auténtico Halal"
    - Button: "Ver Menú" (Yellow background, black text).

### Offers Section
- **Title:** "Ofertas Irresistibles"
- **Grid:** 1 column on mobile, 3 columns on desktop.
- **Cards:** Darker gray background (`#27272a`), rounded corners, subtle shadow.
    - Content: Offer Title (Bold), Price (Red/Yellow), Description.
    - Animation: Fade in + Slide up on scroll.

### Menu Section (Pizzas)
- **Title:** "Nuestras Pizzas"
- **Layout:** List view or 2-column grid.
- **Item Styling:**
    - Name: Bold, Yellow.
    - Ingredients: Italic, Gray.
    - Price: Right-aligned or below, Bold White.

### Contact / Footer
- **Background:** Black.
- **Content:**
    - Address with Map Icon.
    - "Llamar Ahora" Button (Large, Green or Brand Color).
    - Copyright and Halal Badge repeated.

## 3. Interaction
- **Hover Effects:** Buttons scale slightly. Links change color.
- **Scroll:** Smooth scrolling to sections.
- **Entrance:** Elements animate in using Framer Motion.
